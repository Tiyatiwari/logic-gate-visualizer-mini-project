# Logic Gate Visualizer - ECE Mini Project

A React.js based web application to simulate and visualize all 7 basic logic gates: AND, OR, NOT, NAND, NOR, XOR, XNOR. 

Built as a mini-project for Electronics & Communication Engineering.

##  Features
- Real-time logic gate output calculation
- Interactive input toggles for 0 and 1
- Displays truth table for all gates
- Clean and responsive user interface

##  Tech Stack
- **Frontend:** React.js, JavaScript, HTML5, CSS3
- **Package Manager:** npm
